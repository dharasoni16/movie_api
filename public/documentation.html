MyFlix-A movie API

MyFlix is API to build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Movies Endpoints:
Request URL HTTP Method Request Body Data Format Response Body Data Format

", Birth: "May 25, 1972" }, ImageUrl: "https://en.wikipedia.org/wiki/Kabhi_Khushi_Kabhie_Gham...#/media/File:Kabhi_Khushi_Kabhie_Gham..._poster.jpg", Featured: true, }
Get data about a genre by name/title /movies/genre/[genreName] Get None A JSON object holding data about movie genre containing Name and Description,Example { "Name": "melodrama, romance, musical", "Description": "Rahul and Rohan reunite after several years. Sat on a park bench, Rahul asks Rohan questions about things they had talked about when Rohan was just a kid." }
Get data about a director by name /movies/director/[directorName] Get None A JSON object holding data about movie director containing Name, bio,birth year,Example { "Name": "Karan Johar", "Bio": "Karan Johar, often informally referred to as KJo, is an Indian filmmaker and television personality, who primarily works in Hindi cinema. He has launched the careers of several successful actors under his own Dharma Productions.", "Birth": "May 25, 1972" }
Users Endpoints:
Add a user /users Post A JSON object holding data about the users to add, structure like: { name: "John", favoriteMovies: ["Dangal"] } A JSON object holding data about the user that was added, including an ID: { "name": "Jon", "favoriteMovies": [ "Dangal" ], "id": "136cfe8a-e393-43b0-8364-84fe508ba424" }
Update userName for user /users/[id] Put A JSON object holding data about the user that needs to be updated, { "name": "Dhara Updated", "favoriteMovies": [] } A JSON object holding data about the user that was updated, including an ID: { "id": 1, "name": "Dhara Updated", "favoriteMovies": [] }
Add a movie to the list of user favoriteMovies /users/[id]/[movieTitle] Post None A text indicating the movieTitle has been Added
Remove a movie from the list of user favoriteMovies /users/[id]/[movieTitle] Delete None A text indicating the movieTitle has been Removed
Delete a user /users/[id] Delete None A text indicating the user id has been Removed
