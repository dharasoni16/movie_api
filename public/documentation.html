MyFlix-A movie API

MyFlix is API to build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Users Endpoints:
Get all users /users Get None A JSON object holding data about all the users
Get user By Username /users/:Username Get None A JSON object holding data about specified Username, including an ID,For example { "Username": "Jon","Password":"Jon123","Email":"Jon@gmail.com","Birthdate":"1989-01-02", "FavoriteMovies": ["64ecb811852ce60069158c0a"] }
Add a user /users Post A JSON object holding data about the users to add, structure like: For example {"Username": "Jon","Password":"Jon123","Email":"Jon@gmail.com","Birthdate":"1989-01-02" } Note: Username,Password and Email are required fields for users as per define Schema A JSON object holding data about the user that was added, including an ID,For example { "Username": "Jon","Password":"Jon123","Email":"Jon@gmail.com","Birthdate":"1989-01-02" , "favoriteMovies": [], "id": "136cfe8a-e393-43b0-8364-84fe508ba424" }
Update user's details with Username /users/[Username] Put A JSON object holding data about the user that needs to be updated,For example { "Username":"TomUpdated", "Password":"tom345", "Email":"tom@yahoo.com", "Birthdate":"1989-02-01" } A JSON object holding data about the user that was updated, including an ID,For example { "_id": "64ef3e26222c1a282a015dbf", "Username": "TomUpdated", "Password": "tom345", "Email": "tom@yahoo.com", "FavoriteMovies": [], "__v": 0, "Birthdate": "1989-02-01T00:00:00.000Z" }
Add a movie to the list of user favoriteMovies /users/[Username]/movies/[MovieID] Post None A Json object holding userdata including MovieID that has been Added structure like,For example { "_id": "64ef3e26222c1a282a015dbf", "Username": "TomUpdated", "Password": "tom345", "Email": "tom@yahoo.com", "FavoriteMovies": [ "64eca86a852ce60069158c09" ], "__v": 0, "Birthdate": "1989-02-01T00:00:00.000Z" }
Remove a movie from the list of user favoriteMovies /users/[Username]/movies/[MovieID] Delete None A Json object holding userdata without MovieID that has been Removed structure like,For example { "_id": "64ef3e26222c1a282a015dbf", "Username": "TomUpdated", "Password": "tom345", "Email": "tom@yahoo.com", "FavoriteMovies": [], "__v": 0, "Birthdate": "1989-02-01T00:00:00.000Z" }
Delete a user with Username /users/[Username] Delete None A text indicating the Username has been Removed
